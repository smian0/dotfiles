## Portfolio Construction Guidelines

### **70/30 Rule** (Recommended for Most Traders)

**70% Stable (Conservative + Balanced):**
- 50% Conservative Wheel or Dividend Aristocrats
- 20% Mega Cap Tech or Industrial Cyclicals
- **Expected**: 1.3-2.5% monthly blended return
- **Risk**: Low to moderate

**30% High Volatility (Aggressive + Extreme):**
- 20% High Premium Tech or Clean Energy
- 10% Biotech or Quantum (50% position sizing)
- **Expected**: 3-6% monthly on this portion
- **Risk**: High to extreme

**Blended Portfolio Example ($100K)**:
- $50K Conservative ‚Üí $500-1,000/month
- $20K Balanced ‚Üí $300-600/month
- $20K Aggressive ‚Üí $600-1,000/month
- $10K Extreme (50% sizing = $5K) ‚Üí $250-500/month

**Total: $1,650-3,100/month (1.65-3.1%)**
**Annualized: 20-37% with managed risk**

## Preset Customization

### Want to Create Your Own Preset?

Edit `config/scan_presets.py` and add:

```python
'my_custom_preset': {
    'name': 'üéØ My Custom Strategy',
    'description': 'My personal watchlist',
    'risk_profile': 'MEDIUM',
    'expected_premium': '2-4% monthly',
    'beta_range': '1.0-1.5',
    'tickers': [
        'AAPL', 'MSFT', 'GOOGL',  # Add your tickers
        'NVDA', 'AMD', 'TSLA'
    ],
    'settings': {
        'min_discovery_score': 50,
        'signals_required': 2,
        'prefer_small_caps': False,
        'prefer_low_analyst_coverage': False
    }
}
```

Your custom preset will automatically appear in the dashboard!
